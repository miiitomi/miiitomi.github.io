2026年4月19日(日)、茨城県で行われた [**かすみがうらマラソン**](https://www.kasumigaura-marathon.jp/) で初めてフルマラソンを走ってきた。
聞くところによるとなんと人生初フルマラソンというものは人生で一度しか体験できないらしいので、せっかくなのでランニングを始めてからマラソンに挑戦するまでを備忘録的に記してみようと思う。

## 2025年10月：ランニングを始める（6回：17.13km）

31歳になったばかりの2025年10月、急にふとランニングでもしてみようかなと思い立つ。
ランニングを始めたと人に言うと「始めたきっかけor理由は？」ってよく聞かれるのだが、理由としては、①月に1,2度フットサルしてて以前と比べて走れなくなったなと感じフットサルでもっと走れる体力を付けたくなったから、②30代に入って健康・体型維持のため運動不足を解消した方が良さそうに感じたから、③目標を持って取り組める新たな趣味が欲しかったから、の三点くらい。

初めて走った10/23の記録がこちら（後にスマホ/スマートウォッチの設定ミスってて距離が正確に測れてないことが判明したので、この日も実際は表示距離よりもう1,2割ほど長いかも）。

2kmちょっとしか走ってないのに結構しんどかったし、翌日筋肉痛になったの覚えてる。
そのあとは1回3kmで5回ほど続けた。


## 11月：なんとか継続する（14回：66.1km）

急にランニングを始めて継続できるのかと自分でも思ってたけど、意外と継続できた。
11月は14回、大体2日に1回くらいのペースで朝出勤前に走っていた。
この頃は大体3kmから5km走っていて、11月末頃には7km走るようになり、最初2kmで息上がってた頃と比べると1ヶ月でも結構成長してるな〜と達成感を感じていた。
{{< x user=miiitomi id=1995157164630192309 >}}

## 12月：ハーフマラソン挑戦を決意する（14回：126.5km）

1ヶ月2ヶ月と続けてると段々と走れる距離も長くなってきて、だいぶ成長を実感し楽しくなってきた。
そこで12月にはハーフマラソンでも走ってみようかなと決意し、翌月のハーフマラソンに申し込んでみることにした。
そこからはハーフマラソンに向けて走るようになり、長い日は10kmから12km、12月末には15km強走るようになって、月間走行距離は始めて100kmを超えた。
{{< x user=miiitomi id=2006298974337253800 >}}

## 2026年1月：初ハーフマラソン（13回：138.7km）

1月にも日頃のジョギングに加えて週末15km強の長めのランニングを2回ほど行い、1/24に始めてのハーフマラソンに挑戦した。
{{< x user=miiitomi id=2014910675269882133 >}}
初めてのハーフマラソン大会はまず給水に戸惑い、最後は息絶え絶えでかなり苦しかったが、それまでずっと一人で走ってた自分にとって他のランナーさんと一緒に走るのはとても楽しかった。
記録は1時間49分ほど、上位3分の1程度の順位なので始めてにしては結構上出来だったと思う。
周りの人にも早いと褒められ調子に乗り、次はフルマラソンでも挑戦してみるかという気持ちになる。
{{< x user=miiitomi id=2017746473807671418 >}}

## 2月：マラソンに向けてロングランを行う（19回：214.0km）

ハーフマラソン後、次はフルマラソンだということで、1月の時点でシーズン中のうちに走れる大会を探して4月のかすみがうらマラソンに申し込んだ。
この時点での最長走行距離はハーフマラソンの21.1km（練習では17km）、もちろん42kmは未知の世界なので、とにかく長い距離を走れるようになることを最優先目標として練習を続けた。
平日は7kmから10km程度のジョギング（+たまに週1くらいで1km×5本インターバル走または5km閾値走）、週末は土日のどちらかに25kmから30kmくらいのロングラン（5:30~6:00/kmペースくらい）を続けた。
ロングランを行うようになったことで走行距離も伸び、月間走行距離は初めて200kmを超えた。
{{< x user=miiitomi id=2027913891242705244 >}}

## 3月：花粉に苦しむ（21回：252.5km）

3月も2月と同様のメニューで続けていただけのため特に書くことはないのだが、強いて言えば今年の3月~4月頭頃はとにかく花粉が辛かった。
ランナーの皆さん花粉対策どうしてるんですか？誰か助けて。
ちなみに何度走っても30kmランは給水/信号待ち休憩を挟みながらでもめっちゃキツくって、これ本当に42km走れるようになるのかと最後まで不安だった。
{{< x user=miiitomi id=2039289421409947724 >}}

## 4月：初フルマラソン（14回：165.3km）

4月に入ってからは少し練習量を落としてテーパリングを行った上で、4/19のかすみがうらマラソンに臨んだ。
{{< figure
    src="/images/kasumigaura2026/kasumigaura_gate.png"
    width=600
>}}
当日はスタート時点で気温20度を超え、最高で25度近くまで上がるとにかく暑い日だった~~正直なところ初マラソンとしては大会選択ミスったかなと思った~~。
目標はサブ4として、最初から最後まで5:30/kmペースでイーブンで走り切りたいと考えていた。
沿道の応援の多さへの驚きと、スタートから数キロくらい人の多さで思うようなペースで走れないもどかしさを感じながらの初マラソンだったが、高気温のコンディションにも関わらず周りの参加者と一緒に走るためかすごく走りやすく、30kmくらいまでは普段のロングランの練習よりもかなり楽に感じながら、ほぼ設定ペース通り走れた。
練習では30kmまでしか走っていなかったため35kmを超えるとさすがにかなり苦しさを感じたが、それでも沿道の応援のおかげでなんとか最後までペースを落とさずゴールできた。
記録はグロスタイム3:56:02・ネットタイム3:53:26、順位は上位4分の1ほどで目標のサブ4も達成できて大満足。
とても良い思い出となる初フルマラソンだった。

## 半年ランニングを継続してみて

10月にランニングを始めた理由として3つ挙げたが、第一のフットサルで走れるようになるためという点では、以前は1回5分のゲームを数回するとすぐバテていたが、今では2時間のフットサル中でもほぼ最後まで走り切れるようになり、以前以上に最後まで楽しくできるようになった。
第二の健康・体型維持という点では、ランニングを始めてからの半年で体重は5-6kgほど、BMIでいうと22から20くらいまで落ち、周りからも痩せたねとよく言われるようになった。
そして第三の目標に向けて継続できる新しい趣味を持つことという点が一番大きく、目標を実現できた達成感と継続できている充実感とで、メンタル的にとても良い影響があると感じている。

秋からの来シーズンでは次はサブ3.5（3時間半切り）を目標に、今後もランニングを続けていきたい。
あと今は日々一人で黙々とランニングを続けているのでランニング仲間が欲しい。
もしこれからランニング/マラソンを始めてみようと思った人がいたらぜひ教えて。一緒に走りましょう。
